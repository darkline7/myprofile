# myprofile

Static portfolio website for PHAM THANH BINH.

## Run locally

Open `index.html` directly in a browser, or serve the folder with any static server.

## Deploy with cPanel Git Version Control

1. Push this repository to GitHub.
2. In cPanel, open **Git Version Control**.
3. Use the remote URL:

```text
https://github.com/darkline7/myprofile.git
```

4. Set the repository path to your hosting folder, for example:

```text
/home/waiojzlk/public_html
```

5. Pull or deploy the `main` branch.

The site is static, so `index.html`, `style.css`, `script.js`, and `assets/` are enough for hosting.
